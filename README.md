# hamcq

A [Flarum](http://flarum.org) extension. 

## Installation

Install with composer:

```sh
composer require hamcq/share:"*"
```

## Updating

```sh
composer update hamcq/share:"*"
php flarum migrate
php flarum cache:clear
```
